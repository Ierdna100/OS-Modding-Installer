# Installer usage:
```
  -u, --uninstall          Uninstall the OS Modding environnement along with all mods and the mod manager.

  -y                       Answers the equivalent of "yes" to all questions the installer will ask.

  -d, --update             Update the OS Modding environnement along with all mods and the mod manager.

  -i, --check-integrity    Checks the integrity of this installer, the mod manager and the game.

  -p, --platform           Sets the platform for which Doorstop and BepInEx should be installed for. Possible values are
                           'win-x64', 'win-x86', 'macos', 'linux-x64' or 'linux-x86'

  --help                   Display this help screen.

  --version                Display version information.
```
